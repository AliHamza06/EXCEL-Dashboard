Excel Dashboards for eCommerce Dataset

This project contains interactive Excel dashboards analyzing eCommerce data, including sales, orders, and customer behavior.
Dashboards visualize key metrics such as revenue trends, product performance, top customers, and monthly growth.
The data is organized and processed for easy filtering, sorting, and trend analysis using charts and pivot tables.
These dashboards help in making data-driven decisions for marketing, inventory management, and sales strategy.
All visualizations are designed to be user-friendly and dynamic, allowing quick insights without coding knowledge.
